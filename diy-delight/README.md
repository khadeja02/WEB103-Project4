# DIY Delight — Sneaker Studio

A full-stack item personalizer built with React (frontend), Express (backend),
and PostgreSQL (database), hosted on Render.

Users can design a custom sneaker (base model, upper color, sole color, lace
style, size), see the price and a live visual preview update as they choose
options, save designs, and view/edit/delete saved designs later.

See the accompanying chat walkthrough for full step-by-step setup instructions.
Quick start once your `.env` is configured:

```
cd server && npm install && npm run reset-db && npm run dev
cd client && npm install && npm run dev
```
